电力电子技术笔记

Created2022-05-21|Updated2022-05-21
|Post View:

基本概念

电子电力技术是在电子、电力与控制技术基础上发展起来的一门新兴交叉学科。

电子电力技术主要完成各种电能形式的变换，以电能输入输出的形式来分

  • 交流-直流（AC-DC）。该过程一般称为整流
  • 直流-交流（DC-AC）。该过程一般称为逆变
  • 交流-交流（AC-AC）。
  • 直流-直流（DC-DC）。

器件英文缩写：

  • 晶闸管（SCR）

  • 功率二极管（Power Diode）

  • 可关断晶闸管（GTO）

  • 功率场效应晶体管（Power MOSFET）

  • 绝缘栅双极晶体管（IGBT）

电子电力器件的分类：

按照电子电力器件能够被控制电路信号所控制程度分类：

  • 不可控器件

  • 半控型器件

  • 全控型器件

按照开通和关断所需控制极驱动信号不同要求分类：

  • 电流型控制器件（SCR、GTO）
  • 电压型控制器件（IGBT、Power MOSFET）

按照内部电子和空穴两种载流子参与导电的情况分类：

  • 单极型器件（肖特基二极管、Power MOSFET）
  • 双极型器件（功率型二极管、SCR、GTO）
  • 复合型器件（IGBT）

晶闸管：

A：阳极 K:阴极 G：门（栅）极

导通条件：阳极正偏和门极正偏，即UAK>0且UGK>0

主要特性参数：

  • 晶闸管的重复峰值电压——额定电压UT
  • 晶闸管的额定通态平均电流——额定电流IT(AV)
  • 通态平均电压UT(AV)
  • 维持电流IH和擎住电流IL
  • 晶闸管的开通Igt与关断时间tq
  • 通态电流临界上升率di/dt
  • 断态电压裂解上升率du/dt

DC-DC变换器

DC-DC变换器是指能将一定幅值的直流输入电压（或电流）变换成另一幅值的直流输出电压（或电流）的电子电力装置

四种基本类型：

  • 降压型DC-DC变换器，简称Buck变换器
  • 升压型DC-DC变换器，简称Boost变换器
  • 升-降压型DC-DC变换器，简称Boost-Buck变换器
  • 降-升压型DC-DC变换器，简称Buck-Boost变换器

根据是否需要电气隔离分类：

  • 有变压器的隔离型DC-DC变换器
  • 无变压器的非隔离型DC-DC变换器

斩波控制开关管调制规律的不同主要分为两种：

  • 脉冲宽度调制（PWM，即脉宽调制）。这种控制方式是指开关管调制信号的周期固定不变，而开关管导通信号的宽度可调
  • 脉冲频率调制（PFM）。这种控制方式是指开关管导通信号的宽度固定不变，而开关管调制信号的频率可调。

电感电压的伏秒平衡特性：uLtonton + uLtoff toff = 0

电容电流的安秒平衡条件：iCtonton + iCtoff toff = 0

稳态电压增益GV = Uo/Ui

（Ui-Uo）ton = Uo（Ts-ton） 式子（3-3）

令PWM占空比D = ton/Ts,此时GV = Uo/Ui = ton/Ts = D 式子（3-4）

DC-AC变换器

DC-AC变换器是指将一定幅值的直流电变换成一定幅值和一定频率交流电的电力电子装置，又称为逆变器

正弦波为调制波，锯齿波是载波

方波变换时逆变器的交流输出有两种基本调制方式：

  • 脉冲幅值调制（PAM）。采用PAM控制方式，其方波的导通角恒定（180o或120o）
  • 单脉冲调制（SPM）。采用SPM变换方式，逆变输出方波幅值恒定

若令调制波频率为fr，载波频率为fc，则定义N = fc/fr为载波比；若令调制波幅值为Urm，载波幅值为Ucm，则定义M = Urm/Ucm为调制度

异步调制：对于任意的调制频率fr，载波频率fc恒定的脉宽调制称为异步调制

同步调制：对于任意的调制频率fr，载波比N保持恒定的脉宽调制称为同步调制

AC-DC变换器

分析相控整流电路时需熟悉以下几个概念

  • 触发角α
  • 导通角θ
  • 移项
  • 同步
  • 换流

具体内容在课本P162

有源逆变指的是将直流电转换成交流电后，将其返送回电网，这里的源指的是电网

无源逆变器使使电流电路中的电流，在交流侧不与电网连接而直接接入负载(即把直流电逆变为某一频率或可调频率的交流电供给负载)的逆变器。

AC-AC变换器

AC-AC变换器是指能将一种形式的交流电变换成另一种形式的交流电的电力电子变换装置。

根据目标不同，可分为：

  • 交流调压电路
  • 交流电力控制电路
  • 交-交变频电路

软开关变换器

软开关是指在开关电路中增加相应的电感、电容等谐振元件，通过在开关过程引入谐振，使开关管开通前电压先将为零，或关断前电流先降为零，就可以消除开关过程中电压、电流的重叠，从而减小开关损耗。

软开关一般分为零电压开关与零电流开关。

根据软开关技术发展历程又分为**准谐振变换器、 ** PWM软开关变换器

所谓软开关是指开关管同、断过程不存在电压电流交叠区，开关管开通时电压已为零的电压开关ZVS和开关管关断时电流已为零的零电流开关ZCS或近似零电压开关与零电流开关。

所谓硬开关就是开关管开通、关断过程存在电流、电压交叠区。

根据变换器电路中主要开关元件的开关过程是零电压开通还是零电流关断，可以将软开关电路分成零电压电路和零电流电路两大类。
